# Student Management System (C++ + SQLite)

## 📌 Description
A console-based student management system built using C++ and SQLite with full CRUD operations and secure database handling using prepared statements.

## 🚀 Features
- Add Student
- View Students
- Search Student
- Update Student
- Delete Student
- Input Validation
- Persistent Storage

## 🛠️ Technologies
- C++
- SQLite
- Git & GitHub

## ▶️ How to Run
g++ main.cpp student.cpp database.cpp sqlite3.c -o app

./app
